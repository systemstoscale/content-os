## Rotation rule
The agent picks a pillar each day that has NOT been posted in the last 7 days. Default rotation order: S, C, A, L, I, N, G. After all 7 are used, repeat — but never same pillar 2 days in a row.

## When to break the rule
If a pillar's recent post is a hot performer (3x normal engagement), allow re-posting in the same pillar within 7 days with a different angle.
